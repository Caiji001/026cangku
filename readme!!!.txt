V2.0 是研二上学期过年前，调试的最终版本，后面的都是在此基础上改的（是能用的）。没有命令1min后，就会自动进入休眠。发送强制回放全部数据后，可能会读不完就不读了，这是因为就读了1min中的数据，就进入休眠了。这个小问题一般不会影响整体。可以改为10min无命令自动进入休眠（不到8min就全部读完数据）。而且这个强制回放全部数据这个功能不会用到，只有在调试过程中才能用到，上位机读振动数据都是用4A这个命令，所以不必管他，后面的版本暂时都按1min进休眠。每分钟单片机工作前5.0112s（由计算得来具体见程序或汇报PPT），后面55s休眠计时是用的定时器（0.4ms来一个中断，55s/0.4ms=137500）

V2.1 是转过年来，研二下学期，56min的最终版本。改动的地方是每分钟后面55s的计时方式，经试验发现定时器计时的方法，计时太长会导致时间不准（快或者慢几秒），改用RTC时钟计时，是用定时器定时去读RTC的时间，原理就是读取第一分钟采样开始时间，到了第二分钟在开始采样，记得是秒，具体见程序注释（定时器里面），这种方法采样开始时间精准。

V2.2 60min最终版。
